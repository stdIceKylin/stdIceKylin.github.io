CF1942F Solution

Posted on 2025-03-17 Edited on 2025-09-15 In 题解

\(\texttt{CodeForces 1942F}\)

提供一种 \(\mathcal{O}(n+q\log n + q\log\log V)\) 的在线做法。

考虑函数 \(f_i(x) = \left\lfloor \sqrt{f_{i-1}(x) + a_i}\right\rfloor\)，对于两个不同的初始值 \(l, r\)，迭代 \(\mathcal{O}(\log\log V)\) 次后两者之差至多为 \(1\)，利用势能分析是容易证明的。

这启示我们将序列划分成若干长度为 \(B = \mathcal{O}(\log\log V)\) 的块，考虑 \(f_i\) 迭代 \(B\) 次后的函数 \(F_i\)，由上文可知 \(F_i(x) = v_i + [x\ge k_i]\)，其中 \(v_i, k_i\) 的值均可以通过暴力得到。

容易想到所有 \(F_i\) 构成半群 \(\mathcal{F}\)，所以使用支持单点修改，全局半群信息查询的数据结构即可。

注意到最后一个块并非总是完整的，可以将整个序列向后平移使其完整，显然是正确的。
